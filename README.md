Units of measurement for SkeekS CMS
===================================

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/cms-measure "*"
```

or add

```
"skeeks/cms-measure": "*"
```
